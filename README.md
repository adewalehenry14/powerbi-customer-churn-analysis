# E-Commerce Churn Dataset

## Overview
This dataset contains e-commerce customer data used for churn analysis and dashboard building.

## Key Metrics
- Total Customers: 18k
- Churn Rate: 27.4%
- Avg Purchases: 13.6
- Avg Order Value: $122.4
- Avg Lifetime Value: $1,496

## Contents
- Customer demographics (Age Group)
- Purchase history
- Churn status
- Customer value segments (Low/Medium/High)
